## üöÄ About The Project

This repository contains the source code for my personal portfolio website. It serves as a central hub to showcase my experience as a Lead Developer, my technical skill set, and my past projects. 

The design follows a **minimalist dark aesthetic** (`#111319` background) with neon cyan and purple accents, focusing on clean typography, smooth animations, and a unique user experience.

* **üõ°Ô∏è The Gatekeeper:** A unique pre-load overlay that interacts with the visitor before revealing the content (includes a playful check for "idea stealers").

* **üì± Fully Responsive:** Built with mobile-first CSS Grid and Flexbox to look great on all devices.

## üé® Customization

The code is currently contained within a single file structure for portability, but is organized into clear sections.

* **Colors:** Modify the `:root` variables in the `<style>` section to change the theme.
    ```css
    :root {
        --bg-dark: #111319;
        --accent-cyan: #00F5D4;
        --accent-purple: #9B59B6;
    }
    ```
* **Content:** Edit the HTML sections (`#hero`, `#about`, `#projects`, etc.) to update your personal information.
